# TOOLS.md -- Local Environment

## Required Binaries

- `gh` -- GitHub CLI (authenticated)
- `python3` -- Python 3.14+
- `openclaw` -- OpenClaw 2026.4.12+

## Paths

- Skills install to: ~/.openclaw/workspace/skills/<name>/
- Quarantine dir: /tmp/skill-quarantine/
- State dir: <workspace>/data/ (installed-skills.json, search-cache.json, gaps.json, skill-audit.log)
